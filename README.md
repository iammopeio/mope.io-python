# Introduction:

mope.io-python lets you run [Mope.io](https://mope.io) in a Python Script, using CEFPython3.

# Requirements:
  * Python 3.7
  * CEFPython 3
  
# How to run:
  type ```pip install cefpython3``` and type ```mopeio.py``` to Play a game.
